## 프로젝트 소개
이 프로젝트는 (주)한국비전기술에서 제안한 차량 번호판 인식 및 번호 인식이라는 주제를 가지고 YOLOV5를 사용하여 자동차 번호판을 인식하고 ocr 기술을 활용하여 번호판 안의 글자와 숫자을 읽어내는 프로젝트입니다.
